# 文档变更日志

这个文件只用于记录 `ProjectAgents/ProjectAgents.md` 及 `ProjectAgents/references/` 下文档的稳定性更新。

使用规则：
- 只追加，不整理历史，不重写旧记录
- 每次文档更新追加一条，重点写“改了什么”，不要粘贴大段正文
- 记录时间、App、git 提交人、涉及文件和摘要
- 多人协同时如果发生冲突，合并时保留双方记录

模板：

```md
## YYYY-MM-DD HH:MM:SS | AppName | git: user

- Files: `path/a.md`, `path/b.md`
- Summary: 调整了什么
- Summary: 补充了什么
```
